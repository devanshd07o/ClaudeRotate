# 🔄 ClaudeRotate

> **Rotate multiple Claude.ai accounts with one click — auto-inject your custom prompt on every new chat.**

Built by [Devansh Dubey](https://github.com/devanshd07o) · MIT License · Free forever

---

## 💡 What is this?

If you use Claude.ai heavily, you hit usage limits fast.  
ClaudeRotate lets you **cycle through multiple Chrome profiles** (each logged into a different Claude account) with a single double-click — and **auto-sends your custom system prompt** the moment the page loads.

**One click. Next account. Fresh context. Zero friction.**

---

## ✨ Features

- 🔁 Rotates Chrome profiles in order (round-robin)
- 🧠 Auto-injects your custom prompt on every new chat
- ⚙️ One-time setup, works forever after
- 🪶 No API keys. No subscriptions. No browser hacks.
- 💻 Works on any Windows PC

## 🚀 Setup Guide (Do this ONCE)

### Step 0 — Fresh PC? Run Auto-Installer First

> Skip this if you already have Python + Chrome installed.

Double-click **`install.bat`**

It will automatically:
- ✅ Download + install Python if missing
- ✅ Check Chrome is installed (prompts if not)
- ✅ Launch profile setup immediately after

---

### Step 1 — Create Chrome Profiles

1. Open Chrome
2. Click your **profile icon** (top-right circle)
3. Click **"Add"** → Create a new profile
4. Repeat for as many accounts as you want (e.g., Profile 1 to Profile 12)
5. In each profile, go to **https://claude.ai** and **log in** with a different account

> 💡 Tip: Name each profile clearly (e.g., "Claude Account 1")

---

### Step 2 — Download ClaudeRotate

```
Click the green "Code" button → Download ZIP → Extract it anywhere
```

Example: Extract to `C:\ClaudeRotate\`

---

### Step 3 — Run Setup

1. Open the extracted folder
2. Double-click **`setup_profiles.py`** OR open terminal here and run:
   ```
   python setup_profiles.py
   ```
3. It will scan your Chrome profiles and show them:
   ```
   [0] Person 1 → Default
   [1] Claude Acc 1 → Profile 1
   [2] Claude Acc 2 → Profile 2
   ```
4. It will open each profile in Chrome — verify Claude is logged in
5. Enter the numbers of profiles that have Claude logged in:
   ```
   → 0,1,2,3,4
   ```
6. Done! `claude_accounts.json` is saved automatically ✅

---

### Step 4 — Install the Chrome Extension (Auto-Prompt Injector)

1. Open Chrome → go to: `chrome://extensions`
2. Enable **Developer Mode** (toggle, top-right)
3. Click **"Load unpacked"**
4. Select the `extension/` folder inside ClaudeRotate
5. Extension is now active ✅

> This extension auto-types and sends your custom prompt every time `claude.ai/new` opens.

---

### Step 5 — Launch!

Double-click **`Claude.bat`**

Chrome opens → correct profile → claude.ai/new → your prompt auto-sends.  
Next double-click → next account. Automatically.

## 🖥️ Desktop Shortcut Setup (Recommended)

This makes Claude launch silently (no terminal window flash) from a desktop icon.

### Method A — Auto (easiest)

Double-click **`create_shortcut.bat`**  
✅ "Claude" shortcut appears on your Desktop instantly.

### Method B — Manual

1. Right-click Desktop → **New → Shortcut**
2. In the location field, paste this (edit the path to match your folder):
   ```
   C:\Windows\System32\wscript.exe "C:\ClaudeRotate\silent.vbs"
   ```
3. Click **Next** → Name it **Claude** → **Finish**
4. Right-click the shortcut → **Properties → Change Icon** → pick any icon

> 💡 `silent.vbs` runs `Claude.bat` invisibly — no black terminal window pops up.

## 🧠 The Caveman Prompt — What It Does

ClaudeRotate ships with a built-in **token-optimized prompt** that auto-sends on every new Claude chat. It transforms Claude's behavior in 3 ways:

**1. Caveman Communication Mode**
No filler words. No greetings. No apologies. No lengthy explanations.
Claude responds in short, sharp keywords — like a senior dev who respects your time.

**2. Surgical Code/Doc Editing**
Claude never dumps the entire file back at you.
It outputs ONLY the exact lines or blocks that changed — saving tokens, saving time.

**3. Graph-Based Context Navigation**
For large files, Claude first maps your code into a hierarchical tree structure, asks which node to expand, then jumps directly to that location — no re-scanning, no bloat.

> The prompt logic is embedded in the extension. Source is obfuscated to protect the methodology.  
> A customizable version is planned for a future release.

## 🔧 Troubleshooting

| Problem | Fix |
|---------|-----|
| Chrome not found | Edit `chrome_exe` path in `claude_accounts.json` |
| Wrong profile opens | Re-run `setup_profiles.py` |
| Prompt not injecting | Reload extension at `chrome://extensions` |
| Python not found | Re-install Python, check "Add to PATH" |
| Nothing happens on double-click | Run `debug.bat` to see error |

## ❓ FAQ

**Is this legal?**  
Yes. This uses Chrome's built-in multi-profile feature and standard browser automation. No ToS is violated — it's equivalent to manually switching profiles and opening a tab.

**Does this share data between accounts?**  
No. Each Chrome profile is fully isolated — separate cookies, sessions, history.

**Does it work on Mac/Linux?**  
Currently Windows only. Mac/Linux support planned (contributions welcome).

**Do I need to pay for this?**
